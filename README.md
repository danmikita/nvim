## Installation
### Java
1. git clone https://github.com/eclipse/eclipse.jdt.ls.git
2. ./mvnw clean verify
3. Edit the bash script in the lsp/jdtls file to point to your source folder.

### YAML
1. npm install -g yaml-language-server

### Dockerfile
1. npm install -g dockerfile-language-server-nodejs

### Google Java Formatter
1. brew install google-java-format
